# Product Overview

RPG Data Editor (RM数据拓展编辑器) is a desktop application for editing RPG Maker game data files.

## Purpose
A JSON editing tool designed for RPG Maker developers, providing specialized editors for game data with code generation and automatic link management.

## Core Features
- **Script Editor**: Edit RPG Maker scripts with Monaco editor, syntax highlighting, and code completion
- **Property Editor**: Edit game object properties with custom attribute support
- **Note Editor**: Edit game object notes and descriptions
- **Projectile Editor**: Visual editor for projectile animations and trajectories with PixiJS preview
- **Quest Editor**: Complete quest system editor with objectives, rewards, requirements, switches, and variables

## Target Users
RPG Maker game developers who need advanced JSON data editing capabilities beyond the default editor.

## Language
Primary UI language is Chinese (Simplified). Code comments mix Chinese and English.
